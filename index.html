¿Como funcionan las paginas web?

Maximiliano Cano

Sitios Web


Seguramente has oído de los sitios web (siendo que ahora estás en uno), y, probablemente, te hayas sorprendido con la enorme cantidad de diferentes diseños y estilos de todos los sitios, pero... ¿Sábes que se usa para realizarlos? ¿Cómo están hechos? Esta es una pregunta que en estos tiempos les ha surgido a una inmensa cantidad de personas, así que hoy voy a responder esa duda tan grande.

HTML

El HTML es un tipo de código usado para la creación de páginas web. Se basa principalmente en etiquetas que realizan funciones específicas, como mostrar un texto o una imagen. Su uso se ha empezado a reducir con la salida de aplicaciones para crear páginas web, las cuales simplifican el proceso de crearla, pero aunque suenen geniales, limitan las posibilidades ya que están atados a prediseños hechos por el creador. En general el HTML es una gran herramienta, pero es esencial saber codificar ya que no es nada sencillo. Por ejemplo, esta página me tomo alrededor de 12 horas hacer. Así que si no planeas una página web tan grande no es recomendable que dependas de la codificación, puesto que como he mencionado, varias apps ya proporcionan las funciones básicas para un sitio web decente.

CSS

El CSS es un tipo de código utilizado para estilizar los sitios web. Entre sus funciones principales se encuentran el poder agregar imágenes, cambiar el color, cambiar la tipografía, agregar barras y footers, entre muchas más. Al igual que con el HTML, las apps ofrecen (en su mayoria, como caracteristica principal) el poder agregar imágenes, color, fondos, textos, tipografía y demás. En este punto cabe aceptar que la mayoría de apps lo simplifican muy bien, sin quitar demasiadas funciones y con muchas opciones. Esta es una gran elección a la hora de agregar diseño a tus páginas puesto que como se ha mencionado, tiene una gamma alta de opciones para agregar.

JavaScript

El JavaScript es un tipo de código utilizado para que el navegador realice funciones específicas como rellenar un formulario o hacer una susccripción al programa, revista, etc. Este al ser un lenguaje avanzado no se invcluye como tal en las apps, pero estas si cuentan con cierto repertorio de códigos JavaScript muy usados como los formularios. Este código no es muy utilizado pero puede ser de gran ayuda para las páginas mas avanzadas porque da la capacidad de codificar cosas, incluso una app online!
